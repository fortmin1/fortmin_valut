## 关键公式速查

| 公式 | 含义 |
|------|------|
| `P = EPS × P/E` | 股价 = 每股收益 × 市盈率 |
| `市值 = 净利润 × 市盈率` | 公司估值 |
| `F = S × e^(rT)` | 期货定价 |
| `跟踪误差 = 基金收益 - 指数收益` | 基金与指数偏差 |
