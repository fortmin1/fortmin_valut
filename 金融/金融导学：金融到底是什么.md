## 一、经济 vs 金融

| 领域 | 关联对象 | 说明 |
|------|---------|------|
| **经济** | 和政治挂钩 | 政策、制度、政府干预 |
| **金融** | 和法律关联 | 契约、规则、交易条款 |

> 💡 **记忆口诀：** 经济看"政"，金融看"法"

---

## 二、金融产品分类

### 2.1 基础产品（原生资产）

**定义：** 具备所有权的资产

| 类别   | 示例        |     |
| ---- | --------- | --- |
| 股票   | 公司所有权凭证   |     |
| 货币   | 法定货币、外汇   |     |
| 现货商品 | 原油、黄金、农产品 |     |
| 艺术品  | 书画、古董     |     |
| 房产   | 不动产       |     |

**特点：** 交易的是"所有权"

---

### 2.2 金融衍生工具

**定义：** 基于基础资产的合约

| 合约类型     | 说明              |
| -------- | --------------- |
| **远期合约** | 未来某个时间以约定价格买卖   |
| **期货合约** | 标准化的远期合约（交易所）   |
| **期权合约** | 买入或卖出资产的权利（非义务） |
| **互换合约** | 现金流交换（如利率互换）    |

**特点：** 交易的是"合约"本身，不是资产

---

## 三、金融四大特征

| 特征       | 含义             | 风险含义      |
| -------- | -------------- | --------- |
| **联动性**  | 衍生品价格与基础资产联动   | 基础资产波动会放大 |
| **未来合约** | 交易的是未来履约的承诺    | 存在信用风险    |
| **杠杆性**  | 用小资金撬动大交易（保证金） | 收益和损失都被放大 |
| **零和游戏** | 有人赚钱必有人亏钱，总和为零 | 博弈性质强     |

### 图解

```
┌─────────────────────────────────────┐
│           基础产品                   │
│   股票 | 货币 | 商品 | 房产 | 艺术品  │
└──────────────┬──────────────────────┘
               │ 联动性 + 杠杆性
               ↓
┌─────────────────────────────────────┐
│         金融衍生工具                 │
│  远期合约 | 期货 | 期权 | 互换       │
└──────────────┬──────────────────────┘
               │ 未来合约 + 零和游戏
               ↓
        需要法律保障（金融的本质）
```
