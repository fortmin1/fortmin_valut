# 磁盘
## 磁盘分区
ext4 是 **Linux 里最常用的硬盘分区格式**，全称：

**Fourth Extended Filesystem**

第四代扩展文件系统。

你可以把它理解成：

- Windows 里的：**NTFS**
- U 盘里的：**FAT32、exFAT**
- Linux 里的：**ext4**

它们都是**文件系统**，作用只有一个：

**帮你规范地存文件、找文件、删文件。**

硬盘本身只是一堆磁道、扇区，**不知道什么叫文件、目录**。

必须用一种**规则**来管理：

- 文件存在哪
- 占多大空间
- 目录结构怎么存
- 权限、时间戳…

ext4 就是 Linux 下最成熟、最稳定的这套**规则**。
