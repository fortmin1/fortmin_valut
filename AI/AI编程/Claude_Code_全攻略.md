## 第一部分：环境搭建与基础交互

### 1.1 安装 Claude Code
- **时间：** 01:09
- **命令：** `npm install -g @anthropic-ai/claude-code`
- **说明：** 全局安装 Claude Code CLI 工具

### 1.2 登录与授权
- **时间：** 01:43
- **说明：** 首次使用需要进行 Anthropic 账号授权

### 1.3 第一个实战问题
- **时间：** 02:55
- **说明：** 演示如何用自然语言向 Claude Code 提问

### 1.4 三种模式详解
- **时间：** 03:12

| 模式       | 命令/说明                  | 用途            |
| -------- | ---------------------- | ------------- |
| **默认模式** | 标准交互                   | 常规对话和任务       |

| **规划模式** | `--plan` / `Plan Mode` | 先规划再执行，适合复杂任务 |

## 第二部分：复杂任务处理与终端控制

### 2.1 执行终端命令 (Bash)
- **时间：** 06:00
- **用法：** 在 Claude Code 中直接执行 bash 命令
- **示例：** 可以运行 shell 命令、脚本等

### 2.2 使用规划模式 (Plan Mode)
- **时间：** 06:49
- **命令：** `--plan`
- **说明：** 让 Claude 先分析任务并制定执行计划，用户确认后再执行

### 2.3 跳过所有权限检测
- **时间：** 11:06
- **命令：** `--dangerously-skip-permissions`
- **说明：** 跳过所有权限确认提示，用于自动化脚本
- ⚠️ **注意：** 有安全风险，谨慎使用

### 2.4 后台任务管理 (Background Tasks)
- **时间：** 13:53
- **命令：** `--background` 或相关参数
- **说明：** 将长时间运行的任务放到后台执行，不阻塞当前会话

## 第三部分：多模态与上下文管理

### 3.1 版本回滚 (Rewind)
- **时间：** 15:35
- **命令：** `rewind` 或 `/rewind`
- **说明：** 回退到之前的对话状态，恢复到某个历史节点

### 3.2 图片处理
- **时间：** 19:42
- **用法：** 直接在对话中粘贴图片
- **说明：** Claude Code 支持图片输入，可以分析图片内容

### 3.3 安装 MCP Server
- **时间：** 21:09
- **命令：** `claude mcp install <server-name>`
- **示例：** 以 Figma 为例安装 MCP Server
- **说明：** MCP (Model Context Protocol) 扩展 Claude 的工具能力

### 3.4 恢复历史会话 (Resume)
- **时间：** 21:39
- **命令：** `claude sessions resume <session-id>`
- **说明：** 恢复之前中断的会话，继续工作

### 3.5 使用 MCP 工具还原设计稿
- **时间：** 23:06
- **说明：** 通过 MCP 集成第三方工具（如 Figma）还原设计

### 3.6 上下文压缩与清除
- **时间：** 24:49
- **命令：** `/clear` 或相关压缩命令
- **说明：** 清除对话历史，释放上下文空间

### 3.7 项目记忆文件 (CLAUDE.md)
- **时间：** 26:30
- **文件：** `CLAUDE.md`
- **说明：** 在项目根目录创建此文件，Claude Code 会自动读取其中的项目上下文和约定

## 第四部分：高级功能扩展与定制

### 4.1 Hook
- **时间：** 29:46
- **说明：** 钩子机制，在特定时机插入自定义逻辑
- **用途：** 自动化流水线、提交前检查等

### 4.2 Agent Skill
- **时间：** 33:16
- **命令：** `claude skills` 系列命令
- **说明：** 创建可复用的技能模板，定义角色、指令和工具

### 4.3 SubAgent
- **时间：** 36:00
- **命令：** `claude agents` 系列命令
- **说明：** 创建子代理，将复杂任务分配给多个专业代理并行处理

### 4.4 Skill 与 SubAgent 的区别
- **时间：** 39:17

| 特性       | Agent Skill | SubAgent |
| -------- | ----------- | -------- |
| **定位**   | 技能模板        | 独立执行单元   |
| **使用方式** | 被调用         | 可并行运行    |
| **场景**   | 复用最佳实践      | 分解复杂任务   |

### 4.5 Plugin
- **时间：** 40:38
- **说明：** 插件扩展系统，增强 Claude Code 功能

## 📌 常用命令速查表

### 基础命令
```bash
# 安装
npm install -g @anthropic-ai/claude-code

# 启动交互
claude

# 模式切换
claude --plan          # 规划模式
claude --auto          # 自动模式
```

### 会话管理
```bash
claude sessions list    # 列出所有会话
claude sessions resume  # 恢复会话
```

### MCP 相关
```bash
claude mcp install <server>  # 安装 MCP 服务器
claude mcp list             # 列出已安装的 MCP
```

### Skill 与 Agent
```bash
claude skills list         # 列出技能
claude agents list         # 列出代理
```

### 其他
```bash
--dangerously-skip-permissions  # 跳过权限确认
--background                      # 后台任务模式
```
