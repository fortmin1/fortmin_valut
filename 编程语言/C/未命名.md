# Linux生态
## GNU
```
GNU's Not Unix
```
GNU 项目的目标是：

> 做出一套类似 Unix 的、完全自由的软件系统。

它里面有很多经典工具，比如：

```
GCC 编译器套件  
GDB 调试器  
Bash Shell  
make 构建工具  
glibc C 标准库实现  
coreutils 常用命令，如 ls、cp、mv、rm  
binutils 汇编器、链接器等工具
```

```
Linux：内核
GNU：大量系统工具和开发工具
GNU/Linux：Linux 内核 + GNU 用户态工具组成的完整系统
```
